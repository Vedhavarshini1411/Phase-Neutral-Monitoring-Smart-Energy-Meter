# Phase-Neutral-Monitoring-Smart-Energy-Meter

## Overview
This project is an ESP32-based IoT smart energy meter designed to detect electricity theft using phase-neutral current comparison. The system monitors abnormal current variations and provides real-time alerts with relay-based load control.

---

## Features
- Electricity theft detection
- Real-time monitoring
- Relay and buzzer alert system
- IoT-based monitoring
- ESP32-based embedded system

---

## Technologies Used
- ESP32
- Arduino IDE
- IoT Cloud
- Embedded C
- Relay Module
- Buzzer
